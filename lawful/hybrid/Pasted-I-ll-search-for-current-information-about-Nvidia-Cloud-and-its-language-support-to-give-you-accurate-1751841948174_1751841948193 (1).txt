## Language Support for Nvidia Cloud Services

**Direct API Integration**: Nvidia provides REST APIs through their cloud platform that can be accessed from any programming language that supports HTTP requests, including:
- Next.js/React applications
- TypeScript/JavaScript
- Node.js
- Python
- And virtually any other language

**Current Support**: The CUDA parallel computing platform can be programmed with C++, Fortran and Python, but the company is looking for others to run its GPUs. [The New Stack](https://thenewstack.io/nvidia-wants-more-programming-languages-to-support-cuda/
